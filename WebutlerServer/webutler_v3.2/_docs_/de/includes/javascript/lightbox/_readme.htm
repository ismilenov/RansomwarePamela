Lightbox integrieren

Um Bilder in der Lightbox öffnen zu können (Editorfunktion "Bild zoomen") muß das Script
<script src="includes/javascript/lightbox/lightbox_plus.js"></script>
in den Header der Seite eingefügt werden.

Dafür gibt es verschiedene Möglichkeiten:

  1. Beim anlegen einer neuen Seite wird die Zeile von Hand im Quelltext eingetragen.
  2. Man nutzt den Editor-Button "Javascript einfügen".
  3. Man schreibt die Zeile in eines der Layout-Templates zum erstellen neuer Seiten.
  4. Man lädt das Script dynamisch mit PHP Code:
    $webutler_autoheaderdata[] = '<script src="includes/javascript/lightbox/lightbox_plus.js"></script>';
    Diese Zeile kann man z.B. am Ende der globalvars.php eintragen. Das Script wird dann automatisch in jede Webseite geladen.

Wenn die Zeile "Alternativer Text" im "Bild einfügen/bearbeiten" Fenster des Editors ausgefüllt wird, erscheint der Text als Bildbeschreibung unter der Vergrößerung.
