Webfonts - @font-face

In diesem Verzeichnis können verschiedene Webfonts abgelegt werden. Mittels Webfonts können beliebige Schriftarten in HTML-Seiten eingebunden werden (CSS: @font-face{...}). Der Bildeditor greift zur Beschriftung auf dieses Verzeichnis zu und der WYSIWYG-Editor listet die Schriften in der Schriftart-Auswahlbox auf. Dafür müssen die Schriftarten in den folgenden Formaten vorhanden sein.

Der Name der Schriftart muß dem Dateinamen entsprechen:

Beispiel:
Dateiname: oswald_light.ttf
@font-face: font-family:'oswald_light';

Wenn Dateiname und die font-family Angabe nicht identisch sind, kann die Schriftart vom Editor nicht korrekt dargestellt und gesetzt werden.

Dateiformate

TrueType: *.ttf
Embedded OpenType: *.eot
Web Open Font Format: *.woff

Kostenlose Schriftarten in diesen Formaten findet man z.B. bei Font Squirrel.
