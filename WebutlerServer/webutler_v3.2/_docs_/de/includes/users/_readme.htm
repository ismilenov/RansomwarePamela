Modul Users

Das Modul Userpages ist eine kleine Registrierungs- und Loginfunktion für Besucher der Homepage. Als erster Schritt muß das Modul in der Administration installiert und konfiguriert werden. Die Installation ist nur möglich, wenn der Server SQLite3 unterstützt. Nach der Meldung "Datenbank wurde angelegt" sollten unter dem Menüpunkt Einstellungen die Grundeinstellungen vorgenommen werden.

Im Modul können Benutzergruppen erstellt und User in diese Gruppe einsortiert werden. Das bietet die Möglichkeit Seiten für "normale" Besucher zu sperren und nur für einzelne Benutzergruppen frei zu geben. Es können auch einzelne Seitenteile im Quelltext mittels der Gruppen-ID nur für eine Gruppe sichtbar geschaltet werden:

<?PHP if(in_array('GROUPID', $showpartfor)) { ?>
gesperrter Bereich, sichtbar für GROUPID
<?PHP } ?>

Weitere Erklärungen befinden sich im Administrationsfenster.

Loginblock

Als Loginmöglichkeit kann die Datei "loginblock.php" seitenübergreifend als Block includiert werden:

<?PHP include "includes/users/loginblock.php"; ?>

Das Aussehen kann in der "loginblock.css" verändert werden.

Userseiten

Für die Benutzerseiten muß eine neue Seite erzeugt und die Datei "userspage.php" includiert werden:

<?PHP include "includes/users/userspage.php"; ?>

In der Datei /settings/globalvars.php kann ein Extra-Loginlink mit der Variablen $webutler_config['showuserslogin'] ein-/ausgeschaltet werden. Die Gestaltung kann in der "userspage.css" angepasst werden.

Templates für den Loginblock und die Userseiten findet man im Verzeichnis /tpls.
