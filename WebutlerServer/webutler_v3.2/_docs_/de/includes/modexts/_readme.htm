Modulerweiterungen

Die hier hinterlegten Scripte werden in verschiedenen Modulen genutzt.

Datepicker / Kalender

Im /calendar Verzeichnis befindet sich ein kleines Datepicker-Javascript, daß zur Datumsauswahl verwendet werden kann.

PHPMailer

Die PHPMailer Klasse wird zum versenden von eMail-Formularen genutzt.

FPDF

Mit der FPDF-Klasse können PDFs mit PHP erzeugt werden. Im Verzeichnis /tutorial findet man einige Beispiele zur Verwendung von FPDF.

CKEditor

Mit dem CKEditor wird eine Textarea in einen WYSIWYG-Editor umgewandelt. Diese CKEditor Version ist nur eine Basisversion zur Textformatierung, die Ausgabe kann in HTML oder BBCode erfolgen.

CKEditor-Script in den Header laden:

<script src="includes/modexts/ckeditor/ckeditor.js"></script>

Config zur HTML Ausgabe (nach der Textarea setzen):

<script>
/* <![CDATA[ */
CKEDITOR.replace('name-Attribut der Textarea', {
customConfig : 'html_config.js',
language : 'de'
});
/* ]]> */
</script>

Config zur BBCode Ausgabe (nach der Textarea setzen):

<script>
/* <![CDATA[ */
CKEDITOR.replace('name-Attribut der Textarea', {
customConfig : 'bbcode_config.js',
language : 'de'
});
/* ]]> */
</script>
